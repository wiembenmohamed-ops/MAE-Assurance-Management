# Application de Gestion d'Agence d'Assurance - MAE

## Description
Ce projet est une application web développée dans le cadre de mon stage professionnel. Elle permet de gérer efficacement les opérations quotidiennes d'une agence d'assurance MAE, notamment la gestion des clients et des contrats.

## Fonctionnalités
* **Gestion des Clients :** Ajout, modification et consultation de la liste des clients.
* **Gestion des Contrats :** Enregistrement des nouveaux contrats d'assurance.
* **Tableau de Bord :** Visualisation des statistiques (Total clients, contrats et chiffre d'affaires).

## Technologies Utilisées
* **Backend :** PHP
* **Base de données :** MySQL
* **Frontend :** HTML5, CSS3, Bootstrap
* **Environnement :** XAMPP / Localhost
